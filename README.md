# NEXT BYTE Coming Soon Static Website

A single-page static website for the NEXT BYTE coming soon launch. This version keeps the original landing-page structure and updates the branding to match the dark window-wrap artwork: NEXT BYTE, Breakroom at the Rift, bronze linework, off-white type, hex mark, and rift/lightning visual language.

## Files

- `index.html` — page content and structure
- `styles.css` — responsive layout and visual design
- `script.js` — small interactive rift-window effect and current year
- `assets/byte-mark.svg` — simple local logo mark

## Deploy

Upload the full folder to any static host:

- Netlify: drag the folder into Netlify Drop
- Vercel: import as a static project
- GitHub Pages: push the folder contents to a Pages-enabled repo
- Squarespace/Wix/etc.: use as reference for a custom section/page

## Before launch

Replace the placeholder `mailto:` signup form with a real form endpoint such as Mailchimp, Klaviyo, HubSpot, or Formspree.
